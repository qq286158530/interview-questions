# Java 工程师高频算法面试题

> 适用对象：社招跳槽 Java 工程师 | 更新日期：2026-06-01  
> 题目来源：LeetCode 热门题库 + 历年真题规律分析

---

## 【简单】4 道

### 1. LeetCode 1 — 两数之和（Two Sum）
**考察知识点：** 哈希表（HashMap）  
**思路：** 用 Map 记录遍历过的数值及下标，对于每个元素 `nums[i]`，查找 `target - nums[i]` 是否已在 Map 中。一遍遍历 O(n) 解决。

---

### 2. LeetCode 21 — 合并两个有序链表（Merge Two Sorted Lists）
**考察知识点：** 链表、递归、迭代  
**思路：** 维护一个 dummy 头节点，同时遍历两个链表，按大小顺序拼接。也可递归实现，时间 O(n+m)，空间 O(1)（迭代）。

---

### 3. LeetCode 104 — 二叉树的最大深度（Maximum Depth of Binary Tree）
**考察知识点：** 树、DFS（深度优先搜索）  
**思路：** 递归计算左右子树最大深度，取较大者加 1。也可以用 BFS 按层遍历，层数即深度。

---

### 4. LeetCode 70 — 爬楼梯（Climbing Stairs）
**考察知识点：** 动态规划、斐波那契数列  
**思路：** 第 n 阶只能从 n-1 或 n-2 走上来，即 `dp[n] = dp[n-1] + dp[n-2]`，用滚动数组优化至 O(1) 空间。

---

## 【中等】8 道

### 5. LeetCode 15 — 三数之和（3Sum）
**考察知识点：** 双指针、排序、去重  
**思路：** 先对数组排序，固定一个数 `nums[i]`，再用左右指针在 `i+1~end` 区间找两数之和为 `-nums[i]`。重点处理去重，时间 O(n²)。

---

### 6. LeetCode 33 — 搜索旋转排序数组（Search in Rotated Sorted Array）
**考察知识点：** 二分查找  
**思路：** 旋转数组总有一半是有序的。判断 target 落在有序一半还是另一半，不断缩小区间。时间 O(log n)。

---

### 7. LeetCode 206 — 反转链表（Reverse Linked List）
**考察知识点：** 链表、双指针  
**思路：** 迭代：用 prev、curr 双指针依次反转指针方向。递归写法更简洁。掌握两种写法。

---

### 8. LeetCode 146 — LRU 缓存（LRU Cache）
**考察知识点：** 哈希表 + 双向链表 / LinkedHashMap  
**思路：** 用 HashMap 存键值对，用双向链表维护访问顺序（最近使用的移到尾）。O(1) 读写，面试常考手写实现。

---

### 9. LeetCode 200 — 岛屿数量（Number of Islands）
**考察知识点：** DFS/BFS、网格遍历  
**思路：** 遍历二维网格，遇到 '1' 就启动 DFS/BFS 将相邻所有 '1' 标记为已访问，计数器加一。典型"岛屿类"问题模板。

---

### 10. LeetCode 102 — 二叉树的层序遍历（Binary Tree Level Order Traction）
**考察知识点：** BFS、队列  
**思路：** 用队列保存每层节点，一层一层弹出并加入子节点。返回 List<List<Integer>>。可用 BFS 或 BFS 变体（记录层深）。

---

### 11. LeetCode 415 — 字符串相加（Add Strings）
**考察知识点：** 字符串、大数加法  
**思路：** 从字符串末尾同步向前按位相加，用 StringBuilder 收集结果并反转。模拟竖式加法，避免大数溢出。

---

### 12. LeetCode hot100 — 无重复字符的最长子串（Longest Substring Without Repeating Characters）
**考察知识点：** 滑动窗口、双指针、哈希集合  
**思路：** 右指针扩展，维护左指针保证窗口内无重复字符。更新最大长度。时间 O(n)。

---

## 【困难】3 道

### 13. LeetCode 25 — K 个一组翻转链表（Reverse Nodes in k-Group）
**考察知识点：** 链表、模拟、递归  
**思路：** 每 k 个节点为一组进行翻转，不足 k 个不翻转。需要处理组内翻转、组间连接、递归调用三个环节。边界条件多，考验代码鲁棒性。

---

### 14. LeetCode 76 — 最小覆盖子串（Minimum Window Substring）
**考察知识点：** 滑动窗口、哈希表、双指针  
**思路：** 右指针扩展窗口，用计数器记录窗口内各字符出现次数是否满足目标；满足时收缩左指针并更新最小窗口。经典滑动窗口hard题。

---

### 15. LeetCode 42 — 接雨水（Trapping Rain Water）
**考察知识点：** 双指针、动态规划、单调栈  
**思路：** 对于每个位置，能接的雨水量 = min(左边最高, 右边最高) - 当前高度。三种解法（暴力→DP→双指针），双指针最优 O(1) 空间。
